# OpenAPI

<Check>
  Our OpenAPI specification is hosted at
  [https://api.vapi.ai/api-json](https://api.vapi.ai/api-json)
</Check>

## Introduction

Vapi provides a comprehensive OpenAPI specification that describes our API endpoints, request/response structures, and available operations. This specification allows developers to easily integrate Vapi's voice AI capabilities into their applications.

## Accessing the OpenAPI Specification

You can access our OpenAPI specification in JSON format at the following URL:

```
https://api.vapi.ai/api-json
```

This specification is always up-to-date and reflects the latest version of our API.

## Using the OpenAPI Specification

Our OpenAPI specification can be used in various ways:

1. **API Exploration**: Browse the available endpoints, understand request/response structures, and see example payloads.

2. **Code Generation**: Use tools like Swagger Codegen or OpenAPI Generator to create client libraries in your preferred programming language.

3. **Documentation**: Generate interactive API documentation for your team or third-party developers.

4. **Testing**: Create automated tests based on the API specification to ensure your integration works correctly.

## OpenAPI Version

Our specification follows the OpenAPI 3.0 standard, which is widely supported by various tools and libraries in the API ecosystem.
