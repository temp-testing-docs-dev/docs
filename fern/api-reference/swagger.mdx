The Swagger documentation provides a detailed overview of the Vapi API endpoints, request/response formats, and available operations. You can use this interactive documentation to explore and test the API directly in your browser.

## Key Features

- Interactive API documentation
- Request/response examples for each endpoint  
- Try-it-out functionality to test API calls
- OpenAPI/Swagger 2.0 specification

## Accessing the Swagger UI

To access the Swagger UI and explore the Vapi API:

1. Go to [https://api.vapi.ai/api](https://api.vapi.ai/api)
2. You'll see the Swagger UI with all available endpoints grouped by resource
3. Expand an endpoint to see details, parameters, and response formats
4. Use the "Try it out" button to make test API calls

## Using Swagger with Your API Key

To make authenticated API calls via Swagger:

1. Click the "Authorize" button at the top of the page
2. Enter your Vapi API key 
3. Now your API calls will include authentication

## Downloading the OpenAPI Specification

You can download the full OpenAPI specification file to use with other tools:

<Check>
  The OpenAPI JSON file is available at [https://api.vapi.ai/api-json](https://api.vapi.ai/api-json)
</Check>

This allows you to import the Vapi API specification into your preferred API tools and code generators.
